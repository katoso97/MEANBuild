The steps to running the App

  • Create a new folder on your computer
  • Create an account and login to MeanBuild
  • Download Build.zip,place it in your new folder, and unzip it
  • Open the command prompt and run the generator with "node build.js"
  • Answer all questions
    Open Your Text Editor then: 
  • fn 6
    "That will do a javascript build"
    Now go back to Command Prompt and to the following

  • Run this gulp task:
    gulp
    "Run it 2X"
    Run this task to start your local server
  • nodemon
    Now in your browser and go to:
  • localhost:3000


  • That's it you now have a running App
